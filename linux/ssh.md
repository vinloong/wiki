# FAQ
1. no hostkeys available— exiting
   root权限下，重新生成密钥：
	```shell
	ssh-keygen -t dsa -f /etc/ssh/ssh_host_dsa_key
    ssh-keygen -t rsa -f /etc/ssh/ssh_host_rsa_key
	
	chmod 600 /etc/ssh/ssh_host_dsa_key
	chmod 600 /etc/ssh/ssh_host_rsa_key
	```
